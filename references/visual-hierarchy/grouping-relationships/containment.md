# Containment

**Description:**
Using borders or background colors to explicitly group elements together.

**Key Principles:**
- **Bounds**: Defines the edges of content.
- **Internal Consistency**: Elements inside a container share a context.

**Common Patterns:**
- **Cards/Tiles**: The most common form of containment.
- **Modals**: A contained box floating above content.
- **Sidebar**: A darker background containing navigation links.
