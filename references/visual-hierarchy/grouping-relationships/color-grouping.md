# Color Grouping

**Description:**
Using the same color to indicate that items belong to the same category.

**Key Principles:**
- **Coding**: Color-coding can be powerful but requires a legend if not intuitive.
- **Consistency**: If all "Delete" buttons are red, the user learns that red = danger.

**Common Patterns:**
- **Tags/Labels**: "Bug" is red, "Feature" is blue.
- **Calendar Events**: Work events are blue, personal events are green.
