# Alignment Grouping

**Description:**
The Gestalt principle that items aligned along a common edge or axis are perceived as related.

**Key Principles:**
- **Order**: Strong alignment creates a sense of order and relationship.
- **Edges**: Align text to the left (usually) to create a strong vertical axis.

**Common Patterns:**
- **Table Columns**: Data in the same column is related.
- **Grids**: Items aligned to a grid feel like part of a collection.
