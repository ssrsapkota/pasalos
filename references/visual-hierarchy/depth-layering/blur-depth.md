# Blur Depth

**Description:**
Using background blur (backdrop-filter) to create a sense of depth and separation, akin to frosted glass.

**Key Principles:**
- **Legibility**: Ensure text on top of the blur is still readable.
- **Performance**: Heavy use of blur can impact rendering performance.

**Common Patterns:**
- **Glassmorphism**: Translucent cards with background blur.
- **Under-Header**: Header background blurs the content scrolling beneath it.
