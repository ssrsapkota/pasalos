# Shadow Depth

**Description:**
Using drop shadows to emulate elevation and distance from the background.

**Key Principles:**
- **Light Source**: Shadows should generally fall in the same direction (e.g., top-left light source means bottom-right shadow).
- **Soften**: Shadows become blurrier and lighter as the object moves specific "higher".

**Common Patterns:**
- **Elevation Levels**: Material Design's Level 1 (card) to Level 5 (dialog).
- **Colored Shadows**: Using a shadow color that matches the element color for a vibrant look.
