# Overlay Hierarchy

**Description:**
Using semi-transparent overlays (scrims) to direct focus to the top-most layer (like a modal).

**Key Principles:**
- **Dimming**: Darkening the background (`rgba(0,0,0,0.5)`) helps visually recede the background content.
- **Click to Dismiss**: Clicking the overlay should usually close the modal.

**Common Patterns:**
- **Modal Scrim**: Dark background behind a dialog box.
- **Drawer Overlay**: Behind a slide-out navigation menu.
