# Style Contrast

**Description:**
Mixing different visual styles (e.g., serif vs sans-serif, solid vs outlined) to create separation.

**Key Principles:**
- **Pairing**: Combining a clean sans-serif heading with a readable serif body font.
- **Decoration**: Using distinct styles for decorative elements vs functional elements.

**Common Patterns:**
- **Font Pairing**: Roboto (Sans) + Merriweather (Serif).
- **Button Styles**: Primary (Solid) vs Secondary (Outline) vs Tertiary (Text only).
