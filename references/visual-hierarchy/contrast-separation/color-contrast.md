# Color Contrast

**Description:**
The difference in luminance or color that makes an object (or text) distinguishable.

**Key Principles:**
- **Accessibility**: Use WCAG standards (4.5:1 for normal text) to ensure readability for the visually impaired.
- **Attention**: High contrast attracts the eye.

**Common Patterns:**
- **Black on White**: The highest possible contrast (21:1).
- **Complimentary Colors**: High color contrast (e.g., Blue vs Orange).
