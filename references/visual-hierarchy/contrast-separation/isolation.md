# Isolation

**Description:**
Separating an element completely from other elements to give it maximum importance (the "Von Restorff effect").

**Key Principles:**
- **Whitespace**: The primary tool for isolation.
- **Uniqueness**: Making an element look different from everything else (e.g., the only round element on a square page).

**Common Patterns:**
- **Full Page Callout**: A section with a single centered quote or statement.
- **Hero Unit**: Isolated header image.
