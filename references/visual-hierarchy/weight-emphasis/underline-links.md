# Underline Links

**Description:**
The traditional signifier for hyperlinks.

**Key Principles:**
- **Recognition**: Users have been trained for decades that underlined text is clickable.
- **Clutter**: Too many underlines make a paragraph look messy.

**Common Patterns:**
- **Hover Only**: Showing underline only on hover to keep the UI clean (modern style).
- **Always On**: Keeping it always on for maximum accessibility in body text.
