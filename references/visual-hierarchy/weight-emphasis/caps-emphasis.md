# Caps Emphasis

**Description:**
Using ALL CAPS for specific labels or short headings.

**Key Principles:**
- **Letter Spacing**: All caps text needs increased `letter-spacing` (tracking) to be readable.
- **Shouting**: Avoid using it for long sentences, as it looks like shouting.

**Common Patterns:**
- **Eyebrow Headers**: Small, uppercase text above a main headline (e.g., "BLOG POST").
- **Buttons**: Some design systems use caps for button text.
- **Navigation**: Top-level menu items.
