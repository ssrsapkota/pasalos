# Bold Hierarchy

**Description:**
Using font weight (Regular vs Bold vs Extra Bold) to distinguish between headings, body text, and accents.

**Key Principles:**
- **Contrast**: Bold only works if there is non-bold text next to it.
- **Scanning**: Users' eyes jump to bold keywords.

**Common Patterns:**
- **Headings**: Almost always bold.
- **Strong Tag**: Using `<strong>` for key phrases in a paragraph.
- **Labels**: Form labels often bold to stand out from input text.
