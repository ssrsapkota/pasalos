# Focal Point

**Description:**
The specific element that attracts the eye first and holds attention.

**Key Principles:**
- **Singularity**: There should really only be one primary focal point per section. If everything is special, nothing is.
- **Contrast**: Use size, color, or isolation to create the focal point.

**Common Patterns:**
- **Product Shot**: High-quality image of the product.
- **Face**: Humans are wired to look at faces.
