# Z-Pattern

**Description:**
The scanning pattern for pages with less text and more visual elements (like landing pages). The eye moves top-left to top-right, diagonal down to bottom-left, then across to bottom-right.

**Key Principles:**
- **Terminal Points**: Place the Call to Action (CTA) at the end of the Z (bottom right or top right).
- **Flow**: Use visual cues to guide the diagonal movement.

**Common Patterns:**
- **Landing Pages**: Logo (TL) -> Nav/CTA (TR) -> Hero Image -> Headline/CTA (BL/Center).
- **Sign Up Forms**: Input flow.
