# Anchor Points

**Description:**
Visual elements that "ground" the layout and give the eye a place to rest or orient itself.

**Key Principles:**
- **Consistency**: Repeated elements (like a logo or navigation bar) act as anchors across pages.
- **Alignment**: Strong vertical or horizontal lines create anchors.

**Common Patterns:**
- **Logo**: Top-left anchor.
- **Footer**: Bottom anchor.
