# Golden Ratio

**Description:**
Using the mathematical ratio of approx 1.618 to create aesthetically pleasing proportions.

**Key Principles:**
- **Harmony**: It occurs in nature and is perceived as naturally balanced.
- **Grid**: Can be used to determine column widths (e.g., main content 62%, sidebar 38%).

**Common Patterns:**
- **Typography**: Heading size = Body size * 1.618.
- **Layout**: Two-column layouts.
