# Progressive Size

**Description:**
Gradually decreasing size to lead the eye down a path of information.

**Key Principles:**
- **Flow**: Title -> Subtitle -> Body -> Caption.
- **Rhythm**: Consistent steps down in size create a predictable and comfortable reading rhythm.

**Common Patterns:**
- **Article Layout**: Clear progression from H1 down to footer text.
- **Card Design**: Title (L), Description (M), Metadata/Date (S).
