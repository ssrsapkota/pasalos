# Button Prominence

**Description:**
Making the primary call-to-action (CTA) distinctively prominent through size (and color).

**Key Principles:**
- **Touch Target**: Mobile buttons need to be at least 44px height.
- **Hierarchy**: If two buttons are side-by-side, the primary one should often be visually "heavier" or wider.

**Common Patterns:**
- **Full Width**: On mobile, primary buttons often span the full width of the container.
- **Pill Shape**: Large, rounded buttons for friendly, prominent CTAs.
