# Headline Dominance

**Description:**
Ensuring that headings effectively structure the content and draw the eye first.

**Key Principles:**
- **Scannability**: Users scan headings to find relevant sections.
- **Inverted Pyramid**: The main headline (H1) should be the largest, followed by subsections (H2), etc.

**Common Patterns:**
- **Display Fonts**: using a distinctive font for headlines vs body text.
- **Tight Leading**: Large text usually needs tighter line-height.
