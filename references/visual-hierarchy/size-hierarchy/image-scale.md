# Image Scale

**Description:**
Using the size of images to dictate their importance in the narrative.

**Key Principles:**
- **Hero Image**: A large, full-width image sets the mood and is the primary visual anchor.
- **Thumbnails**: Small uniform images suggest a list of equal-value items.

**Common Patterns:**
- **Grid Layout**: Varying image sizes (masonry) to break monotony and highlight feature items.
- **Avatar Sizing**: Large profile pic on profile page, small avatar in comments.
