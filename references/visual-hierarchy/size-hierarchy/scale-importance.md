# Scale Importance

**Description:**
The principle that larger elements are perceived as more important. Size is the most powerful tool for establishing hierarchy.

**Key Principles:**
- **Ratio**: Use a consistent typographic scale (e.g., 1.25 or 1.5 ratio) so sizes relate harmoniously.
- **Dramatics**: significant size differences create drama and clear focal points.

**Common Patterns:**
- **Hero Text**: Massive headlines to announce the main value prop.
- **Primary vs Secondary**: Primary buttons are often larger (or at least same size but bolder) than tertiary links.
