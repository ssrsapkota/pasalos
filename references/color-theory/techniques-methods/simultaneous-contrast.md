# Simultaneous Contrast

**Keywords:** Illusion, perception, trickery, context.

**Description:**
The phenomenon where a color looks different depending on the colors surrounding it.

## Usage

### 1. The Effect
*   A "Gray" square on a "Black" background looks lighter.
*   The same "Gray" square on a "White" background looks darker.

### 2. UI Implication
You cannot trust the hex code to look the same everywhere.
*   **Fix:** You might need two different hex codes for "Secondary Text" depending on if it's on a white card or a light-gray body background.

### 3. Vibrating Edges
Placing two highly saturated colors next to each other creates a jarring, vibrating line.
*   **Fix:** Add a neutral border (White or Black) between conflicting colors to separate them.
