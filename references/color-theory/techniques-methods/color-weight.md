# Color Weight

**Keywords:** Balance, gravity, attention, size.

**Description:**
Different colors have different "visual weights". This affects how "heavy" or "large" an element feels on the page.

## Usage

### 1. The Physics
*   **Darker** colors weigh more than **Lighter** colors.
*   **Saturated** colors weigh more than **Muted** colors.
*   **Red** is heaviest; **Yellow** is lightest.

### 2. Balancing a Layout
If you have a large block of *Light Gray* (Low weight), you can balance it with a small button of *Dark Blue* (High weight).
*   **Big & Light** = **Small & Heavy**.

### 3. Typography
*   A Bold font in a light color (Gray) might have the same visual weight as a Regular font in a dark color (Black).
