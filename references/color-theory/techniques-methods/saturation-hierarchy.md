# Saturation Hierarchy

**Keywords:** Focus, depth, attention, clean.

**Description:**
Using the intensity of color (chroma) to define importance, rather than just changing the hue.

## Usage

### 1. The Saturation Scale
*   **Primary Action (Highest Saturation):** The button you want clicked.
*   **Headlines (Medium Saturation):** Important information.
*   **Body Text (Low Saturation):** Dark grays/blues.
*   **Backgrounds (Lowest Saturation):** Near-white or near-black tints.

### 2. The "Graying Out" Effect
We intuitively understand that desaturated things are "older", "disabled", or "background".
*   Use this to push secondary content back without hiding it.

### 3. Pro Tip
If a screen feels "messy", it often has too many competing saturation levels. Reduce the saturation of everything except the 1 key action.
