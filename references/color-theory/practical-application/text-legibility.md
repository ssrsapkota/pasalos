# Text Legibility

**Keywords:** Reading-level, scanability, strain, type-color.

**Description:**
Ensuring text is comfortable to read for long periods.

## Usage

### 1. The 60-30 Rule
*   **Headings:** Can be lower contrast or colorful (Short reading time).
*   **Body:** Must be high contrast and neutral (Long reading time).

### 2. Pure Black is Sharp
`#000000` text on a white background is the highest contrast possible, but it can cause eye strain (jitter).
*   **Recommendation:** Use Dark Gray (`#111827`) for a softer, more premium reading experience.

### 3. Colored Text
If you must use colored text (e.g., a Blue link):
*   Ensure it is dark enough to pass WCAG AA.
*   `#3B82F6` (Standard Blue 500) fails on white.
*   `#2563EB` (Blue 600) passes.
