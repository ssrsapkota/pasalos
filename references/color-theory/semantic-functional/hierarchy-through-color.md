# Hierarchy Through Color

**Keywords:** Importance, scanning, attention, visual-weight.

**Description:**
Using color intensity to guide the user's eye to the most important elements first. Not everything can be "Primary Blue".

## Usage

### 1. Primary Action (The "Call to Action")
*   **Quantity:** 1 per view (or section).
*   **Color:** The strongest, most saturated brand color. Solid fill.
*   **Goal:** "Do this main thing."

### 2. Secondary Action
*   **Quantity:** Multiple allowed.
*   **Color:** Outline (Ghost) or Desaturated/Tinted fill.
*   **Goal:** "Alternative options" (Cancel, Back, Save Draft).

### 3. Tertiary Action
*   **Quantity:** Unlimited.
*   **Color:** Text link styling or simple icon. No background.
*   **Goal:** "Navigation or minor details."

### 4. Backgrounds
*   **White:** Content area.
*   **Light Gray:** Grouping/containers.
*   **Color:** Featured/Hero area only.

**Rule:** If everything is bold, nothing is bold.
