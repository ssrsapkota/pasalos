# Neon & Cyber (Cyberpunk)

**Keywords:** Future, tech, dystopia, electric, glowing.

**Description:**
Hyper-saturated colors against deep dark backgrounds. "Light" is a key material here, often simulated with glows (box-shadows) and gradients.

## Usage

### 1. Key Colors
*   **Cyber Pink:** `#ff00ff`
*   **Cyan:** `#00ffff`
*   **Electric Green:** `#00ff00`
*   **Background:** Deep Navy or Black `#050510`

### 2. The Glow Effect
A neon color is just a flat color without a glow.
*   **Technique:** Use `box-shadow` or `drop-shadow` with the same color as the element to create luminosity.
*   `box-shadow: 0 0 10px #00ffff;`

### 3. Application
*   Crypto, Gaming, Web3, Nightlife.
