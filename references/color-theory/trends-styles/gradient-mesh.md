# Gradient Mesh (Aurora)

**Keywords:** Ethereal, modern, blurred, abstract.

**Description:**
Instead of a linear start-to-end gradient, multiple points of color are placed on a canvas and blurred together, creating fluid, organic shifts.

## Usage

### 1. Tech Stack
*   **CSS:** `filter: blur(100px)` on absolute positioned circles.
*   **Canvas:** WebGL fluids.

### 2. Color Selection
Pick 3-4 Analogous colors (e.g., Blue, Purple, Pink) and 1 highly contrasting Accent (e.g., Neon Yellow) to create "hot spots".

### 3. Application
*   SaaS backgrounds (Stripe style).
*   Hero sections where photography is unavailable.
*   "Living" backgrounds that animate slowly.
