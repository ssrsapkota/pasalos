# Muted & Soft (Pastel/Earthy)

**Keywords:** Sophisticated, vintage, organic, gentle.

**Description:**
Colors that have been desaturated (mixed with gray) or tinted (mixed with white). These colors feel "grown-up" and are easier to look at for long periods than high-saturation colors.

## Usage

### 1. Creating Muted Tones
Don't pick from the top-right of the color picker. Move towards the left (desaturate).
*   Instead of #FF0000 (Red), use #D9534F (Dusty Red).
*   Instead of #0000FF (Blue), use #5B9BD5 (Steel Blue).

### 2. The "Instagram" Aesthetic
Popular in lifestyle, fashion, and artisanal e-commerce. It conveys authenticity and calmness.

### 3. Typography
Avoid pure black text. Use Dark Gray (#333) or Dark Blue (#1e293b) to match the softness of the palette.

```css
.soft-card {
  background-color: #fdfbf7; /* Off-white */
  color: #4b5563; /* Gray-600 */
  border-left: 4px solid #d4a373; /* Muted Gold */
}
```
