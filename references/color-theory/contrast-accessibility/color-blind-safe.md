# Color Blind Safe

**Keywords:** Deuteranopia, Protanopia, Tritanopia, inclusivity, shapes.

**Description:**
Designing palettes that are distinguishable by people with color vision deficiencies (approx 8% of men). relying on hue alone to convey meaning is the number one cause of accessibility failures.

## Usage

### 1. The Red/Green Trap
Red and Green look identical (brownish-yellow) to Deuteranopes.
*   **Fix:** Use Blue and Orange instead. Blue/Orange is the safest contrasting pair for almost all vision types.
*   **Fix:** Use Red (Darker value) and Green (Lighter value).

### 2. Double Coding
Never use color alone. Always pair color with a symbol or label.
*   **Bad:** A red dot for error, green dot for success.
*   **Good:** Red dot with "X" icon or text "Error". Green dot with Checkmark.

### 3. Texture and Pattern
Use patterns (stripes, dots) in charts to differentiate data series instead of just solid colors.

### 4. Tools
*   **Simulate:** Use "Rendering" tab in Chrome DevTools to simulate color blindness.
*   **Palette Builders:** Adobe Color has a "Color Blind Safe" filter.
