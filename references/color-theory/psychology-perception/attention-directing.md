# Attention Directing (The Pop)

**Keywords:** Focus, hierarchy, squish-test, conversion.

**Description:**
Using color as a tool to physically move the user's eyeball to a specific spot on the screen.

## Usage

### 1. The Von Restorff Effect
The item that looks different is the one that is remembered.
*   If your page is 90% Blue, a tiny Orange button will get 100% of the attention.
*   If your page is 90% Orange, an Orange button is invisible.

### 2. Isolation
Color works best when surrounded by whitespace or neutrals.
*   **Dont:** Put a bright button on a bright background.
*   **Do:** Put a bright button on a gray/white background.

### 3. Eye Pathing
Use the same distinct color for steps in a wizard (1 -> 2 -> 3) to drag the eye across the page linearly.
