# Color Memory

**Keywords:** Brand-recall, distinctiveness, proprietary.

**Description:**
Humans are terrible at remembering exact hues, but great at remembering relationships.

## Usage

### 1. Own the Color
*   **Coke:** Red.
*   **Tiffany:** Robin's Egg Blue.
*   **Home Depot:** Orange.
*   *Goal:* Be so consistent that the color alone signals the brand.

### 2. Limitation
We can only distinguish about 5-7 colors in short-term memory.
*   *UI Rule:* Don't use more than 5 distinct hues in your dashboard if you expect users to remember what each one means (Semantic Load).

### 3. Consistency
If "Edit" is Blue on page 1, it must be Blue on page 100. Changing it to Green breaks color memory and forces relearning.
