# Analogous

**Keywords:** Natural, harmonious, serene, comfortable.

**Description:**
Groups of three colors that are adjacent to each other on the color wheel. One color is typically dominant, while the others are used to enrich the scheme. Usually matches commonly found relationships in nature (e.g., blue, blue-green, green).

## Usage

### 1. Selection Strategy
*   **Dominant:** Choose your main brand color (e.g., Green).
*   **Support 1:** One step clock-wise (Blue-Green).
*   **Support 2:** One step counter-clock-wise (Yellow-Green).

### 2. Distribution (60-30-10)
*   60% Dominant (Backgrounds, headers).
*   30% Support 1 (Secondary buttons, cards).
*   10% Support 2 (Accents, icons).

### 3. Application Example
For a finance app:
*   **Base:** Teal (Trust/Growth).
*   **Analogous 1:** Blue (Stability) for headers.
*   **Analogous 2:** Green (Money) for success states.

```css
/* Serene Nature Palette */
.theme-forest {
  --color-moss: #4a6741;      /* Dominant */
  --color-fern: #3d8c40;      /* Analogous 1 */
  --color-lime: #88a73b;      /* Analogous 2 */
}
```
