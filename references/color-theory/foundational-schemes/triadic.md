# Triadic

**Keywords:** Balanced, colorful, playful, dynamic.

**Description:**
Three colors that are evenly spaced around the color wheel. This tends to be quite vibrant, even if you use pale or unsaturated versions of your hues.

## Usage

### 1. Hierarchy is Key
To use a triadic harmony successfully, the colors should be carefully balanced. Let one color dominate and use the two others for accent.
*   **Dominant:** 60%
*   **Accent 1:** 30%
*   **Accent 2:** 10%

### 2. Common Triads
*   **Primary:** Red, Yellow, Blue (Very energetic, childish, or "superhero").
*   **Secondary:** Purple, Orange, Green (More sophisticated, quirky).

### 3. Application
Great for dashboards or illustrations requiring distinct categories that are equal in weight but different in meaning.

```css
/* Dashboard Categories */
.chart-segment-1 { background: #E74C3C; } /* Red */
.chart-segment-2 { background: #F1C40F; } /* Yellow */
.chart-segment-3 { background: #3498DB; } /* Blue */
```
