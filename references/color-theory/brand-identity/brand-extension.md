# Brand Extension

**Keywords:** Scale, tints, shades, Tailwind, system.

**Description:**
Taking a client's single logo color and turning it into a full design system (scale of 50-900).

## Usage

### 1. The problem
A single color `#0066CC` is not enough for a UI. You need lighter versions for backgrounds and darker versions for text/hover.

### 2. The Scale Technique
*   **50-200:** Light tints (Mix with white). Used for backgrounds.
*   **300-400:** Medium tints. Used for borders.
*   **500:** The Brand Color. Used for Buttons.
*   **600-700:** Dark shades (Mix with black/hue-shift). Used for Hover states.
*   **800-900:** Deep shades. Used for Text.

### 3. Hue Shifting (Pro Tip)
Don't just add black/white. Shift the hue towards the "warm" or "cool" side as you change lightness to keep colors looking fresh.
*   *Lighter Red* -> Shift towards Orange (Salmon).
*   *Darker Red* -> Shift towards Purple (Wine).

### 4. Tools
*   Tailwind Color Generator.
*   LCH (Lightness-Chroma-Hue) pickers.
