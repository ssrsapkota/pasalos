# Color Personality

**Keywords:** Psychology, brand-values, marketing, alignment.

**Description:**
Matching color choices to the attributes of the brand.

## Usage

### 1. The Archetypes
*   **Red:** Aggressive, Bold, Energetic (Coke, Netflix, Target).
*   **Blue:** Trustworthy, Secure, Corporate (Chase, Facebook, IBM).
*   **Green:** Growth, Healthy, Money (Whole Foods, Android, Fidelity).
*   **Yellow:** Optimistic, Caution, Cheap (McDonalds, Best Buy).
*   **Purple:** Royals, Creative, Spiritual (Twitch, Cadbury).
*   **Black:** Luxury, Exclusive, Serious (Apple, Nike, Mercedes).

### 2. Misalignment Risk
A "Security Bank" using Neon Pink will feel untrustworthy.
A "Kids Toy" using Desaturated Brown will feel depressing.

### 3. Application
Start the project by listing 3 brand adjectives. Pick the hue that overlaps the most with those adjectives.
