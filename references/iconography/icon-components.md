# Icon Components

> **React/Vue icon systems**

Wrapping SVGs in framework components for maximum reusability and developer experience.

## Benefits
-   **Props:** easy API for size, color, stroke width. ` <Icon name="home" size={24} />`
-   **Consistency:** Enforces default sizes and styles.
-   **Code Splitting:** Modern bundlers (bundlers?) can simple import specific icons to avoid bloating the bundle. `import { IconHome } from './icons'`.

## Libraries
-   **Lucide:** Excellent open-source React icon set.
-   **Heroicons:** Tailwind-friendly.
-   **Phosphor:** Flexible weight system.
