# Abstract Icons

> **Geometric representations**

Used when no physical object exists to represent the concept, or for branding/logo purposes.

## Examples
-   **Logos:** Brand marks are usually abstract.
-   **Data Concepts:** Connections, nodes, AI, algorithms often use abstract geometry (dots, lines, networks).
-   **Status:** Colors (Red/Green/Yellow) or shapes (Circle/Triangle/Square) indicating state.

## Challenges
-   Hardest to learn.
-   Requires heavy reliance on onboarding, labels, and consistent context to create meaning for the user.
