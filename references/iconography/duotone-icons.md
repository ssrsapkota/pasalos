# Duotone Icons

> **Two-tone depth and dimension**

Duotone icons use two colors (or two shades of the same color) to add depth, hierarchy, and visual interest. They bridge the gap between flat icons and illustrations.

## Characteristics
-   **Two Layers:** A primary layer (usually darker/opaque) for the main shape, and a secondary layer (lighter/transparent) for details or fill.
-   **Opacity:** Often uses opacity (e.g., 100% stroke, 30% fill) to create the two tones using a single color value.
-   **Depth:** Creates a pseudo-3D effect or shadowing.

## Usage
-   **Feature Highlights:** Empty states, onboarding screens.
-   **Dashboards:** Adding visual richness to data-heavy interfaces.
-   **Modern Brands:**  Tech and SaaS products often favor this style.

## Best Practices
-   Ensure sufficient contrast between the two tones.
-   Keep the lighter tone subtle so it doesn't compete with the defining strokes.
-   Consistent application of which elements get the secondary tone.
