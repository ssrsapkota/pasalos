# Corner Radius

> **Consistent rounded corners**

The radius of corners defines the "personality" of the icon set—sharp is technical/serious, rounded is friendly/playful.

## Guidelines
-   **Sharp (0px):** Precise, technical, architectural.
-   **Soft (1-2px):** Modern, standard UI friendly.
-   **Round (Full):** Playful, bubbly.

## Consistency
-   Apply the same radius rule to all outer corners.
-   Inner corners generally do not need rounding unless the outer radius is very large.
-   **Concentric Corners:** If you have parallel lines (lines inside a shape), the inner radius should determine the outer radius (Inner Radius + Stroke Width = Outer Radius) to look mathematically correct.
