# Filled Icons

> **Solid shapes, bold presence**

Filled icons use solid shapes to define objects. They have a higher visual weight than outlined icons, making them distinct and easily recognizable.

## Characteristics
-   **Solid Areas:** Shapes are completely filled with color.
-   **Silhouette Based:** Relies on the recognizability of the object's silhouette.
-   **High Contrast:**  Creates strong definition against backgrounds.

## Usage
-   **Active States:** Often used to indicate an active state in a navigation bar (e.g., Outline for inactive, Filled for active).
-   **Primary Actions:** FABs (Floating Action Buttons), primary call-to-actions.
-   **Emphasis:** When an icon needs to stand out.

## Best Practices
-   Ensure the silhouette is clear and not cluttered.
-   Use negative space carefully to define internal details (e.g., the keyhole in a lock).
-   Pair with outlined icons for state changes (Active/Inactive) if appropriate for the system.
