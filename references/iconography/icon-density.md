# Icon Density

> **Sizing for compact vs spacious UI**

Density determines how tightly packed information is. Icons play a big role in perceived density.

## Spacious (Consumer/Marketing)
-   Use larger icons (24px+).
-   Generous padding (12px-16px).
-   Communicates ease, friendliness, and focus.

## Compact (Enterprise/Data Tools)
-   Use smaller icons (16px-20px).
-   Tighter padding (4px-8px).
-   Communicates efficiency, power, and maximum data visibility.

## Principles
-   Keep the icon container size consistent even if the internal icon visual size changes.
-   Don't mix densities on the same screen (e.g., don't put a giant airy icon in a tight data grid).
