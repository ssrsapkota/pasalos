# Icon Only

> **When icons can stand alone**

Icon-only buttons save space and reduce visual clutter, but they rely entirely on recognition.

## Safe Usage
-   **Universal Metaphors:** Search, Home, Close, Back.
-   **Contextual Clarity:** A "Trash" icon next to a list item item is clearly "Delete".
-   **High Frequency:** Standard controls (media players: Play, Pause, Next) don't need labels.

## Requirements
-   **Tooltips:** ALWAYS provide a tooltip on hover for desktop users.
-   **Accessibility:** ALWAYS provide an `aria-label` or `sr-only` text for screen readers.
