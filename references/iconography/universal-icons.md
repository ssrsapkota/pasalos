# Universal Icons

> **Globally understood symbols**

Universal icons bypass language barriers and are understood by users across different cultures and age groups without explanation.

## Examples
-   **Magnifying Glass:** Search.
-   **House:** Home.
-   **Gear:** Settings.
-   **Printer:** Print.
-   **X:** Close/Cancel.
-   **Check:** Confirm/Success.

## Usage
-   Use these safely without text labels if space is tight.
-   Don't try to reinvent the wheel for these concepts; familiarity breeds usability.
