# Metaphor Icons

> **Real-world object references**

Skeuomorphic roots. Using physical objects to represent digital concepts helps users build a mental model.

## Examples
-   **Trash Can:** Deletion (Temporary holding place).
-   **Folder:** Organization container.
-   **Bell:** Notifications (Ringing for attention).
-   **Envelope:** Email/Messages.

## Evolution
-   Metaphors evolve. The "Floppy Disk" for Save is a classic example of a "dead metaphor" that is still universally understood despite the object being obsolete.
-   Be careful with new metaphors; ensure the physical object is recognizable to your demographic.
