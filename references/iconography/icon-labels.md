# Icon Labels

> **When to pair with text**

Text labels act as a safety net for icons, removing ambiguity and reducing cognitive load.

## When to Label
-   **Navigation Bars:** Bottom tabs usually benefit from labels to clarify sections.
-   **Complex Actions:** Specialized tools (e.g., "Merge", "Rebase", "Distribute Horizontal") almost always need tooltips or persistent labels.
-   **New/Novel Icons:** If the symbol isn't universal, label it.

## The 5-Second Rule
-   If it takes a user more than 5 seconds to guess what an icon does, it needs a label.
