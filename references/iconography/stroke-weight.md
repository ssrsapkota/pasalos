# Stroke Weight

> **Matching type weight (1.5px, 2px)**

Consistent stroke weight is crucial for a cohesive icon set and for harmonizing with typography.

## Guidelines
-   **Standard Weight:** A 2px stroke on a 24px icon is a common robust standard.
-   **Light Weight:** A 1.5px or 1px stroke creates a lighter, more elegant feel.
-   **Scaling:** When scaling icons, decide if strokes should scale (for illustrative purposes) or remain constant (for UI consistency).
-   **Type Pairing:**
    -   Ex: 2px stroke pairs well with Medium/Semibold text.
    -   Ex: 1.5px stroke pairs well with Regular text.

## Implementation
-   Use centered strokes in vector tools for easier resizing, but inner/outer strokes for pixel-perfect control.
-   Avoid mixing multiple stroke weights in a single icon unless it's a specific stylistic choice (e.g., Duotone).
