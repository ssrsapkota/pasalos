# Icon Scale

> **12, 16, 20, 24, 32, 48px sizes**

Establishing a set of standard sizes ensures hierarchy and consistency across the UI.

## Standard Sizes
-   **12px:** Micro. Inline with very small text, or complex data indicators. Hard to read.
-   **16px:** Small. System utilities, dense lists.
-   **20px:** Secondary. slightly smaller than standard, often used in dense toolbars.
-   **24px:** Standard. The default size for most touch interactions and primary navigation.
-   **32px:** Large. Feature icons, section headers.
-   **48px+:** Hero/Display. Marketing, empty states, splash screens.

## Scaling Strategy
-   **Don't just resize:** Scaling a 24px icon down to 12px usually results in a blurry mess.
-   **Optimize:** Create specific optimized versions for smaller sizes (removing detail) and larger sizes (adding detail).
