# Card Variants

## Common Layouts

### Vertical (Stacked)
- **Standard**: Image top, Content middle, Footer bottom.
- **Usage**: Blog posts, product cards, user profiles.
- **Constraints**: Usually fixed width or follows a grid column width.

### Horizontal (Row)
- **Standard**: Image left, Content right.
- **Usage**: News feeds, list items with improved visual richness.
- **Responsive**: Often stacks vertically on mobile screens.

### Compact / Mini
- **Standard**: Icon + Title + Value.
- **Usage**: Dashboard stats, feature highlights.
- **Style**: Minimal padding, smaller typography.

### Image Overlay
- **Standard**: Background image covering the full card, text overlay with gradient protection.
- **Usage**: Immersive visual content, portfolios, galleries.
- **Accessibility**: Ensure text contrast is sufficient over dynamic images (use scrims/overlays).

### Action / Call-to-Action (CTA)
- **Standard**: Focused message with a prominent button.
- **Style**: May use a distinct background color or border to stand out from content cards.
