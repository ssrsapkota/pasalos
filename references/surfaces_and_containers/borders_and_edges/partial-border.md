# Partial Border

## Directional Definition
Using borders on only specific sides of an element.

### Usage
- **Bottom Only**: Common for mobile list items or sticky headers.
- **Left/Right Only**: Common for indicating a quote, a "callout" block, or the active state in a sidebar navigation item (a "stripe").
- **Between Items**: Using `divide-y` or `divide-x` utilities to place borders *between* children but not on the outer container.

### Visual Weight
- A single colored border on one side (e.g., `border-l-4 border-l-blue-500`) is a strong signal of category or status without requiring a full background color change.
