# Border Color

## Contrast & Subtlety
Border color defines the "hardness" of a container's edge.

### Palettes
- **Subtle**: Low contrast (e.g., `gray-200` on white). Use for structural division that shouldn't distract.
- **Prominent**: Medium contrast (e.g., `gray-400`). Use for inputs or actionable areas.
- **Interactive**: Brand color (e.g., `blue-500`). Use for focus states or selected items.
- **Error**: Semantic color (e.g., `red-500`). Use for validation errors.

### Dark Mode
- Borders in dark mode often need to be darker/more subtle than you think (`gray-800` or `gray-700`).
- High contrast borders in dark mode can look harsh and "neon".
