# Spacing Divider

## Whitespace as Separator
Using negative space to define relationships without visible lines.

### Principles
- **Closeness = Relationship**: Items close together are related. Items far apart are separate sections.
- **Rhythm**: Consistent spacing (e.g., `gap-8`, `my-12`) creates a predictable flow.

### Implementation
- **Margins/Padding**: `py-16` or `my-20`.
- **Grid Gap**: `gap-y-12`.
- **Containers**: Using separate background colored containers implicitly creates a divider where they meet.
