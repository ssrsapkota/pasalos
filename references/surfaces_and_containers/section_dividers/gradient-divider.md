# Gradient Divider

## Color Fade Transitions
Soft, seamless blending between sections.

### Styles
- **Fade to Transparent**: Main content fades out into the background color.
- **Gradient Line**: A 1px line that fades from transparent -> color -> transparent.
- **Section Blend**: A large gradient overlay (`bg-gradient-to-b`) connecting two different background colors.

### Usage
- Good for long-scrolling landing pages.
- avoiding harsh lines creates a more immersive, "storytelling" feel.
