# Image Background

## Visual Storytelling
Using photography or illustration to set the mood.

### Considerations
- **File Size**: Optimize images (WebP/AVIF).
- **Responsive**: `object-cover` and `background-position` are critical.
- **Resolution**: Provide `@2x` assets for high-DPI screens.

### Overlays
- **Darkening**: `bg-black/50` overlay to ensure white text pops.
- **Blur**: Blurring the background image (`backdrop-filter`) helps text readability.
- **Gradient Fade**: Fading the image into a solid color at the bottom to transition to content.
