# Pattern Background

## Repeating Textures
Adding structure without high cognitive load.

### Styles
- **Geometric**: Dots, grids, lines, hexagons.
- **Organic**: Waves, scribbles, noise.
- **Subtle**: Low contrast (e.g., light gray dots on white) allows patterns to be used on large surfaces.

### Implementation
- **SVG Patterns**: crisp and scalable.
- **CSS Background Repeat**: `background-image: url(...)`.
- **Masking**: Using a pattern as a mask for a gradient.
