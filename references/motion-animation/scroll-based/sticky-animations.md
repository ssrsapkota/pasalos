# Sticky Animations

Elements that pin to the viewport then animate or transform.

### Mechanics
- **Property**: `position: sticky`.
- **Orchestration**:
    - Item sticks to top.
    - As user continues scrolling, the *stuck* item changes (scales down, changes color, content inside it scrolls horizontally).
- **Use Case**:
    - Dynamic headers.
    - "Scrollytelling" features.
