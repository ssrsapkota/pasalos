# Button Feedback

Communicating state through the primary action element.

### States
1.  **Hover**: Slight lift (shadow) or brightness increase.
2.  **Press**: Scale down (e.g., 0.95 or 0.98) to mimic physical button deforming.
3.  **Loading**: Text disappears, spinner appears (or button morphs into spinner).
4.  **Disabled**: Opacity reduced, cursor `not-allowed`.
