# Shake Error

Universal signal for "something is wrong".

### Mechanics
- **Motion**: Rapid x-axis translation (e.g., -10px, +10px, -8px, +8px...).
- **Timing**: Fast interactions (e.g., 300-400ms total).
- **Use Case**:
    - Incorrect password.
    - Form submission validation failure.
    - Action denied.
