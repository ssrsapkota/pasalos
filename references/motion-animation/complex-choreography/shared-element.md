# Shared Element

Persisting specific elements across route changes.

### Mechanics
- **Identity**: Assign a unique ID (`layoutId` in Framer Motion).
- **Behavior**: The element appears to "fly" from its old position/size to its new position/size.
- **Use Case**:
    - Product card image -> Product details page header image.
    - Avatar in list -> Avatar in profile header.
- **Goal**: Context persistence.
