# Gesture Driven

Motion that directly responds to user touch/drag.

### Mechanics
- **Inputs**: `drag`, `swipe`, `pinch`.
- **Physics**: Velocity, friction, rubber-banding.
- **Direct Manipulation**: 1:1 movement with the finger.
- **Use Case**:
    - Swipe to dismiss.
    - Pull to refresh.
    - Draggable ordering.
- **Goal**: Tangible, physical interface feel.
