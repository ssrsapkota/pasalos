# Scale In

Growing an element from a smaller point.

### Mechanics
- **Property**: `transform: scale(0.9 to 1)`.
- **Use Case**: Modals, popovers, badges, FAB (Floating Action Button).
- **Tip**: Avoid scaling from 0 unless using a spring effect, as it can look like it's "sucking" into existence. Scaling from 0.9 or 0.95 is cleaner.
