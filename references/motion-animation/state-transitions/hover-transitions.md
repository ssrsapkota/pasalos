# Hover Transitions

Indicating interactivity and inviting a click.

### Mechanics
- **Properties**: `transform` (scale, translateY), `box-shadow`, `background-color`.
- **Timing**: Fast (e.g., 200ms `ease-out`).
- **Goal**: Make the interface feel "alive" and responsive to the cursor.
- **Example**: Card lifts slightly (`translateY(-4px)`) and shadow grows darker/softer on hover.
