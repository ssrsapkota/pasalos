# Disabled Transitions

Visually indicating that an element cannot be interacted with.

### Mechanics
- **Property**: `opacity` (0.6 or 0.5), `filter: grayscale(100%)`.
- **Cursor**: `not-allowed`.
- **Transition**: Should be smooth if state changes dynamically (e.g., form becomes invalid).
- **Goal**: Prevent frustration by clearly showing unavailability.
