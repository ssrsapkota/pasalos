# Content Swap

replacing one piece of data with another in the same container.

### Mechanics
- **Old Content**: Fade out + subtle slide/scale out.
- **New Content**: Fade in + subtle slide/scale in.
- **Layout**: Might require `position: absolute` during transition to prevent layout jumps.
- **Use Case**:
    - Tab switching.
    - Carousel slides.
    - Filtering results.
