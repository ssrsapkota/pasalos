# Essential Motion

Animation that conveys critical information.

### Characteristics
- **Purpose**: Communicates state change, relationship, or feedback that *must* be understood.
- **Example**:
    - Error shake (indicates failure).
    - Focus ring movement (indicates keyboard position).
    - Progress bar (indicates system status).
- **Compliance**: Allowed even under "reduced motion" if subtly executed, or must have a static alternative.
