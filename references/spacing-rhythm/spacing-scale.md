# Spacing Scale

**Standard:** 4px base scale

The 4px grid is the foundation of our spacing system. It ensures consistency and rhythm across the UI.

## The Scale

| Token | Value | Rem | Usage |
|-------|-------|-----|-------|
| 4     | 4px   | 0.25| Micro spacing, tight groupings |
| 8     | 8px   | 0.5 | Icon/text gaps, tight padding |
| 12    | 12px  | 0.75| Small component padding |
| 16    | 16px  | 1   | Standard component padding |
| 24    | 24px  | 1.5 | Section spacing, loose padding |
| 32    | 32px  | 2   | Major component separation |
| 48    | 48px  | 3   | Layout sections |
| 64    | 64px  | 4   | Major layout sections |
| 96    | 96px  | 6   | Hero sections, distinct breaks |
| 128   | 128px | 8   | Page sections, distinct breaks |

## Usage Guidelines
- Always stick to the scale. Avoid arbitrary values like 11px or 13px.
- Use smaller values for related items (grouping).
- Use larger values for separating distinct sections.
