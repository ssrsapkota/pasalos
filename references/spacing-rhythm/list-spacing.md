# List Spacing

**Concept:** Gaps between list items.

Proper spacing in lists improves scanability.

## Types
- **Bullet Lists:** 
  - Gap between items: `mb-2` or `space-y-2`.
  - Line height: 1.5 or 1.6.
- **Data Lists (Key: Value):**
  - Need clear separation between rows.
  - Borders often help, with `py-4` padding.
- **Card Lists (Grid):**
  - Gap between cards: `gap-6` (24px) is standard.

## Visual Grouping
If a list item has multiple lines (e.g., title + subtitle), ensure the gap *within* the item is smaller than the gap *between* items.
