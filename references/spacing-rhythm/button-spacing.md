# Button Spacing

**Concept:** Internal and external gaps.

## Internal (Padding)
- Buttons need to look clickable.
- **Height:** Defined by vertical padding + line height.
- **Width:** Horizontal padding should usually be 2x the vertical padding (e.g., `px-4 py-2`). This creates an elegant proportion.

## External (Gap)
- **Primary + Secondary:** When two buttons sit side by side (e.g., "Save" and "Cancel"), use a consistent gap (usually 12px or 16px).
- **Group:** In a button group (joined buttons), the border is shared, but the touch target remains standard.
