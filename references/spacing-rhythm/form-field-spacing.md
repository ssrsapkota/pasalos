# Form Field Spacing

**Concept:** Label, input, helper gaps.

Forms require strict vertical rhythm to guide the user through the completion process.

## The Unit
1. **Label:** Title of the input.
2. **Gap 1:** Small (e.g., 8px).
3. **Input:** The field itself.
4. **Gap 2:** Small (e.g., 8px) for helper text/error.
5. **Gap 3:** Large (e.g., 24px) before the NEXT field unit starts.

## Principle
The Label must clearly belong to the Input spacing-wise. If the gap between the Label and Input is the same as the gap between the previous Input and the current Label, the user gets confused.
