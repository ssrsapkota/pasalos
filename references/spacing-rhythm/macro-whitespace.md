# Macro Whitespace

**Concept:** Large gaps between sections and major layout containers.

Macro whitespace organizes the overall layout of the page. It tells the user where one topic ends and another begins.

## Usage
- **Section Breaks:** Significant padding between specialized sections (e.g., Features vs Testimonials). Often 64px, 96px, or even 128px.
- **Gutters:** The space between columns in a grid.
- **Margins:** The outer margins of the main content container.

## Impact
- **Pacing:** Controls the rhythm of the user's reading experience. Large gaps slow the user down, allowing them to digest information.
- **Structure:** Clearly delineates the architecture of the page.
