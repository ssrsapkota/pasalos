# Asymmetric Space

**Concept:** Intentional imbalance.

While symmetry brings stability, asymmetry brings energy and movement. Using asymmetric spacing can create more dynamic and interesting layouts.

## Techniques
- **Offset Layouts:** One column is wide, the other narrow, creating tension.
- **Broken Grid:** Elements overlap or break out of strict alignment columns.
- **Randomized Spacing:** Subtle variations in spacing to mimic organic arrangements (controlled chaos).

## Use Cases
- Editorial designs.
- Brand storytelling pages.
- "Layout Rebellion" concepts.
- To guide the eye in a specific diagonal path across the screen.

**Warning:** Requires a strong understanding of balance. Unintentional asymmetry just looks broken.
