# Density Modes

**Concept:** Compact for data, relaxed for reading.

Responsive Spacing isn't just about screen size; it's about context and device capability.

## Compact Mode
- **Context:** Professional tools, data grids, mouse-driven desktop usage.
- **Characteristics:** tighter padding, smaller fonts, maximum data per pixel.

## Relaxed Mode
- **Context:** Tablets, consumption apps, news readers.
- **Characteristics:** Airy, open, focus on ease of reading and touch interaction.

## Implementation
You can apply a class to the `body` (e.g., `density-compact` or `density-comfortable`) that re-defines your CSS variables for spacing tokens.
