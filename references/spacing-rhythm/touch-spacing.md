# Touch Spacing

**Concept:** Larger gaps for touch interfaces.

Fingers are less precise than mouse cursors. Interactive elements on touch devices need larger hit areas and safer separation.

## Targets
- **Min Size:** Apple recommends 44x44pt. Android recommends 48x48dp.
- **Safe Zone:** Ensure there is enough empty space around a button so the user doesn't accidentally click the neighbor.

## Hybrid Web
Since most modern web apps run on both, default to "Touch Friendly" spacing. It rarely hurts desktop users to have slightly larger, easier-to-click buttons.
