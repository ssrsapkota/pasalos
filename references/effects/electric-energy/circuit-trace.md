# Circuit Trace

**Keywords:** PCB, Motherboard, Path, Cyber, Flow.

**Description:**
lines of light travelling along 90-degree or 45-degree paths, resembling data moving on a circuit board.

## Usage

### 1. SVG Path Animation
Animate the `stroke-dashoffset` of a path drawn with orthogonal lines.

```css
.circuit-line {
  stroke-dasharray: 20 100;
  animation: flow 2s linear infinite;
}
```
