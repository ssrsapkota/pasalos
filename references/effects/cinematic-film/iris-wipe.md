# Iris Wipe

**Keywords:** Looney Tunes, Close, Circle, Transition, End.

**Description:**
A black circle closing in until the screen is fully black (iris in), or opening up (iris out).

## Usage

### 1. CSS Clip Path
`clip-path: circle(100% at 50% 50%)`.
Animate radius to `0%`.

### 2. SVG Mask
Detailed method allowing for feathered edges using a radial gradient mask.
