# Sun Rays

**Keywords:** God Rays, Sunshine, Radiation, Brightness.

**Description:**
Bright streaks of light emanating from a central source, often used to simulate the sun or a holy light.

## Usage

### 1. CSS Conic Gradient
A repeating conic gradient can create radial stripes.

```css
.sun-rays {
  background: repeating-conic-gradient(
    from 0deg,
    rgba(255, 255, 0, 0.2) 0deg,
    rgba(255, 255, 0, 0.2) 10deg,
    transparent 10deg,
    transparent 20deg
  );
  animation: spin 60s linear infinite;
}
```

### 2. SVG
Lines radiating from a center point.
