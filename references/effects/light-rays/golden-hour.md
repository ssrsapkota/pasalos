# Golden Hour

**Keywords:** Sunset, Warmth, Orange, Soft, Directional.

**Description:**
The warm, soft, directional light found shortly after sunrise or before sunset. Characterized by long shadows and gold/orange hues.

## Usage

### 1. Overlay & Blend Mode
Apply a warm orange/yellow gradient overlay with `mix-blend-mode: overlay` or `soft-light`.

```css
.golden-hour-overlay {
  background: linear-gradient(120deg, rgba(255, 160, 0, 0.4), rgba(255, 100, 0, 0.1));
  mix-blend-mode: soft-light;
  pointer-events: none;
}
```
