# Laser Beam

**Keywords:** Sharp, Neon, Focused, Sci-Fi, Scanner.

**Description:**
A very thin, incredibly bright line of colored light, often with a "bloom" or glow around the core.

## Usage

### 1. CSS Box Shadow
A 1px or 2px div with a strong colored box-shadow.

```css
.laser {
  height: 2px;
  background: white;
  box-shadow: 
    0 0 5px #f00,
    0 0 10px #f00,
    0 0 20px #f00; /* Red Laser */
}
```
