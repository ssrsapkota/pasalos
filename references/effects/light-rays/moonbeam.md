# Moonbeam

**Keywords:** Night, Cool, Blue, Silver, Soft.

**Description:**
A cool, soft, bluish-white light often associated with moonlight.

## Usage

### 1. CSS Gradients
Similar to Golden Hour but with cool colors (blues, cyans, silvers).

```css
.moonbeam-gradient {
  background: radial-gradient(circle at top right, rgba(200, 220, 255, 0.3), transparent 60%);
}
```
