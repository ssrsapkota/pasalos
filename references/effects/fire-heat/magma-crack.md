# Magma Crack

**Keywords:** Floor, Damage, Lava, Fissure, Heat.

**Description:**
Glowing cracks appearing on a surface, pulsing with heat.

## Usage

### 1. CSS Clip Path or SVG Mask
Reveal an underlying "lava" div using a jagged clip-path or SVG mask.

```html
<!-- Lava Layer -->
<div class="lava-glow animate-pulse"></div>
<!-- Overlay with holes -->
<div class="ground-overlay" style="mask-image: url('cracks.svg')"></div>
```
