# Match Strike

**Keywords:** Ignite, Flash, Sulphur, Spark, Start.

**Description:**
A sudden bright flash of white/yellow light, followed by a small settling flame.

## Usage

### 1. CSS Animation Sequence
Script a class change on click.
1. `.flash`: scale(2), huge brightness, duration 0.1s.
2. `.flame`: grow from scale(0) to scale(1), orange.

```css
.flash {
  background: white;
  box-shadow: 0 0 50px white;
  opacity: 1;
}
```
