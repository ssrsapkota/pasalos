# Noise Flow

**Keywords:** Perlin Noise, Simplex Noise, Flow Field, Generative Art.

**Description:**
Particles or lines moving along varying paths generated by a noise algorithm (like Perlin noise). This creates natural, wind-like, or fluid-like motion that looks random but is mathematically coherent.

## Usage

### 1. Canvas + Simplex Noise
You generally need a noise library (e.g., `simplex-noise` or `open-simplex-noise`).

```javascript
// Minimal Concept
const canvas = document.getElementById('noise-canvas');
const ctx = canvas.getContext('2d');
// ... setup size ...

const zOff = 0; // Time dimension

function draw() {
   // Loop through a grid of points
   for (let x = 0; x < width; x += 20) {
       for (let y = 0; y < height; y += 20) {
           // Get noise value for this coordinate
           const angle = noise.simplex3(x * 0.01, y * 0.01, zOff) * Math.PI * 2;
           
           // Draw line vector based on angle
           const length = 10;
           ctx.beginPath();
           ctx.moveTo(x, y);
           ctx.lineTo(x + Math.cos(angle) * length, y + Math.sin(angle) * length);
           ctx.stroke();
       }
   }
   zOff += 0.005; // Move through time
}
```

### 2. SVG Turbulence (Static/Texture)
For static "noisy" textures.
```html
<filter id="noise">
  <feTurbulence type="fractalNoise" baseFrequency="0.65" numOctaves="3" stitchTiles="stitch" />
</filter>
```
