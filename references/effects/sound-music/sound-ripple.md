# Sound Ripple

**Keywords:** Echo, Location, Ping, Wave, Spread.

**Description:**
Concentric rings fading out from a source, representing sound travelling through air.

## Usage

### 1. CSS Border
[See water-liquid/water-ripple.md](../water-liquid/water-ripple.md).
Key difference: Sound ripples are usually faster and more uniform than water ripples.

```css
@keyframes sound-wave {
  0% { width: 0; opacity: 1; }
  100% { width: 300px; opacity: 0; }
}
```
