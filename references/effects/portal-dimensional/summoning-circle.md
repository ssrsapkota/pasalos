# Summoning Circle

**Keywords:** Magic, Rune, Glow, Rotation, Ritual.

**Description:**
Concentric rings of runes and geometric shapes rotating in opposite directions, glowing intensely.

## Usage

### 1. SVG & CSS Rotation
Create 3-4 rings of patterns in Illustrator.
Export as SVG.
Animate each ring rotating at different speeds and directions using CSS.

```css
#ring-outer { animation: spin 20s linear infinite; }
#ring-inner { animation: spin 10s linear infinite reverse; }
```
