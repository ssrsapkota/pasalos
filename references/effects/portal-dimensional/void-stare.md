# Void Stare

**Keywords:** Abyss, Dark, Eyes, Scary, Infinite.

**Description:**
A pitch black area that seems to look back at the user, possibly with faint red eyes or tentacles in the darkness.

## Usage

### 1. CSS Vignette & Pulse
Extreme vignette (radial gradient black at edges) that pulses inward slowly.
Occasional faint glow animation at the center.

```css
.void {
  background: radial-gradient(circle, transparent 20%, black 90%);
  animation: breathe 5s infinite alternate;
}
```
