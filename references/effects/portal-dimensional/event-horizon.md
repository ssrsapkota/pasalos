# Event Horizon

**Keywords:** Edge, Boundary, Glow, Danger, Physics.

**Description:**
The glowing edge around a black hole or portal where light is trapped.

## Usage

### 1. CSS Box Shadow & Border
A circular div with a very specific shadow stack.

```css
.horizon {
  background: black;
  box-shadow: 
    inset 0 0 20px 10px #f50,
    0 0 10px 2px #f50;
  border-radius: 50%;
}
```
