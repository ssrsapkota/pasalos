# Transformation

**Keywords:** Morph, Shape-shift, Poof, Change, Evolution.

**Description:**
A character changing form, usually hidden by a cloud of smoke or light to mask the model swap.

## Usage

### 1. The "Poof" Trick
Spawn a large [Smoke Plume](../smoke-vapor/smoke-plume.md) or [Flash Bang](../impact-explosion/flash-bang.md) to cover the object.
Swap the DOM element or 3D model while it is obscured.
Fade smoke.
