# Levitation Glow

**Keywords:** Fly, Lift, Hover, Feet, Anti-gravity.

**Description:**
A steady glow or jet of energy underneath a character's feet or an object to indicate it is hovering.

## Usage

### 1. CSS Box Shadow (Inset)
If hovering over ground: A shadow that gets smaller and sharper as the object rises.
If emitting energy: A glow that pulses on the object's bottom edge.
