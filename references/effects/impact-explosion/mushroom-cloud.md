# Mushroom Cloud

**Keywords:** Nuke, Explosion, Big, Cloud, Rising.

**Description:**
A large explosion that rises into a cap shape with a trailing column.

## Usage

### 1. Particle System (Fluid Simulation)
Advanced visualization.
Simpler version: Spawn a large group of particles that move up.
The top particles spread out horizontally and slow down.
The bottom particles (column) stay narrow.
Color: Start bright yellow/white, fade to black/grey smoke.
