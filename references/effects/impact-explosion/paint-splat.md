# Paint Splat

**Keywords:** Color, Art, Messy, Gooey, Liquid.

**Description:**
Identical to [Blood Splatter](blood-splatter.md) but with bright, varied colors and higher viscosity (thicker, rounder drops).

## Usage

### 1. SVG Filter (Gooey)
Merge multiple circles using the contrast/blur filter trick to simulate a single thick liquid mass.
