# Ground Crack

**Keywords:** Fissure, Earthquake, Split, Break, Damage.

**Description:**
A jagged line opening up on the ground, often revealing lava or darkness below.

## Usage

### 1. Mask Reveal
Have a "cracked" texture layer hidden behind a mask.
Animate the mask opening up along the path of the crack.

### 2. Canvas Path
Draw a jagged path that grows over time. Increase line width to "open" it.
