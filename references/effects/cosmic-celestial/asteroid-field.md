# Asteroid Field

**Keywords:** Debris, Rocks, Floating, Tumbling, Space.

**Description:**
Many irregular 3D objects floating and rotating randomly in space.

## Usage

### 1. CSS 3D
Use `preserve-3d` and rotate various polygon shapes (or images of rocks).

### 2. Canvas
Draw irregular polygons (generated by randomly shifting vertices of a circle).

```javascript
// Generate Asteroid Shape
ctx.beginPath();
for (let i = 0; i < numVertices; i++) {
  let angle = (i / numVertices) * TWO_PI;
  let r = radius + random(-5, 5); // Jaggedness
  ctx.lineTo(cos(angle) * r, sin(angle) * r);
}
ctx.closePath();
```
