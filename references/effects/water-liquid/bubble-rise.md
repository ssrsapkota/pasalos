# Bubble Rise

**Keywords:** Carbonation, Underwater, Air, Floating, Sphere.

**Description:**
Transparent spheres rising around with a slight wobble.

## Usage

### 1. Particles
Similar to [Ember Float](../fire-heat/ember-float.md) but with a different visual style (outline, highlight).

```css
.bubble {
  border: 1px solid rgba(255,255,255,0.5);
  background: radial-gradient(circle at 30% 30%, rgba(255,255,255,0.8), transparent);
  border-radius: 50%;
  animation: rise 4s ease-in infinite;
}
```
