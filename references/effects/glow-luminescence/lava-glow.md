# Lava Glow

**Keywords:** Magma, Heat, Molten, Red, Orange.

**Description:**
Deep red/black surface with cracks of bright orange/yellow glowing oozing through.

## Usage

### 1. CSS Overlay & Masking
Use a "cracked earth" PNG texture as a mask over a moving orange/red gradient.

### 2. Canvas Noise
Perlin noise where values > threshold are black (rock) and values < threshold are orange (lava).
