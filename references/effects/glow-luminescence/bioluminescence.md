# Bioluminescence

**Keywords:** Organic, Avatar, Sea, Blue, Pulse.

**Description:**
Soft, pulsing blue/green light coming from living organisms or organic shapes.

## Usage

### 1. CSS Animation
A subtle pulse animation on blue/green elements.

```css
@keyframes bio-pulse {
  0%, 100% { box-shadow: 0 0 10px #00ffaa; opacity: 0.8; }
  50% { box-shadow: 0 0 25px #00ffaa; opacity: 1; }
}
```

### 2. Canvas
Particles that slowly fade in and out with perlin noise movement.
