# Neon Glow

**Keywords:** Cyberpunk, Tube Light, Electric, Vapourwave.

**Description:**
Bright, saturated colored lines with a soft outer glow, simulating neon gas tubes.

## Usage

### 1. CSS Box/Text Shadow
Layer multiple shadows with increasing blur radius.

```css
.neon-text {
  color: #fff;
  text-shadow:
    0 0 5px #fff,
    0 0 10px #fff,
    0 0 20px #ff00de,
    0 0 30px #ff00de,
    0 0 40px #ff00de,
    0 0 50px #ff00de,
    0 0 60px #ff00de;
}
```
