# Blacklight UV

**Keywords:** Ultraviolet, Fluorescence, Club, Purple, Reactive.

**Description:**
Deep purple/blue background light where specific neon colors (greens, pinks) pop intensely.

## Usage

### 1. CSS Mix Blend Mode
Overlay a purple layer with `mix-blend-mode: multiply` but keep the neon elements on top or use `mix-blend-mode: screen`.

### 2. Color Palette
Use a very dark purple (`#1a0033`) background and high-saturation neon accent colors.
