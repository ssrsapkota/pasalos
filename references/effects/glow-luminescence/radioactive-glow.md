# Radioactive Glow

**Keywords:** Token, Green, Hazardous, Pulse, Aura.

**Description:**
A sickly, vivid green pulse or aura, often associated with hazardous materials.

## Usage

### 1. CSS Filter
Use `drop-shadow` with a specific lime green color.

```css
.radioactive {
  filter: drop-shadow(0 0 10px #39ff14);
  animation: rad-pulse 2s infinite;
}
```
