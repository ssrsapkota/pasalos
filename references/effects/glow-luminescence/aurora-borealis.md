# Aurora Borealis

**Keywords:** Northern Lights, Curtain, Flow, Green, Sky.

**Description:**
Waving curtains of light in the sky, usually greens and purples.

## Usage

### 1. CSS Filter & Keyframes
See `light-glow/aurora-gradient.md` for a detailed implementation using blurred blobs.

### 2. Canvas
Draw multiple sine waves with varying amplitude and phase, filled with a vertical gradient that fades to transparent at the bottom.
