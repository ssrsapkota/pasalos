# Boot Sequence

**Keywords:** Terminal, Console, Startup, BIOS, Log.

**Description:**
Lines of technical text appearing rapidly one after another, scrolling up.

## Usage

### 1. JS Intervals
Append `<p>` tags to a container container with `overflow: hidden`.
Auto-scroll to bottom.
Use `typing` animation for the last line for extra effect.

```css
.log-line {
  font-family: monospace;
  color: #0f0;
}
```
