# Pixel Dissolve

**Keywords:** Disintegrate, Voxel, Thanos, blocks, Fade.

**Description:**
An object breaking apart into square pixels that drift away.

## Usage

### 1. Canvas / JS Access
Read image data, create an array of "Pixel" particles with corresponding colors, then animate them moving with random velocity/gravity.

### 2. CSS Houdini (Paint API)
Custom paint worklet to draw pixels.
