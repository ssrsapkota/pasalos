# Sonar Ping

**Keywords:** Submarine, Echo, Ring, Expand, Locate.

**Description:**
[See Pulse Wave](../electric-energy/pulse-wave.md).
Often used on maps to show location.
Differs from Radar Sweep as it is omnidirectional.

## Usage

### 1. CSS Animation
Multiple expanding rings with `border`.
Key aspect: The ring should fade out as it expands to simulate signal loss.
