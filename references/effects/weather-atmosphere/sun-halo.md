# Sun Halo

**Keywords:** Sundog, Ice, Ring, Bright, Atmospheric.

**Description:**
A faint ring of light around the sun, often caused by ice crystals in the atmosphere.

## Usage

### 1. CSS Box Shadow
A large, incredibly blurred white/rainbow box shadow.

```css
.sun-halo {
  box-shadow: 
    0 0 50px 20px rgba(255, 255, 255, 0.4),
    0 0 100px 50px rgba(255, 200, 100, 0.1);
}
```
