# Morning Mist

**Keywords:** Dawn, Haze, Ground, Soft, Calm.

**Description:**
A very low, static layer of white haze that sits just above the ground.

## Usage

### 1. CSS Gradient
A linear gradient from transparent to semi-transparent white at the very bottom of the screen.

```css
.mist {
  background: linear-gradient(to top, rgba(255,255,255,0.4), transparent);
  height: 20vh;
  position: absolute;
  bottom: 0;
}
```
