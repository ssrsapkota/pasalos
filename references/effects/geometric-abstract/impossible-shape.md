# Impossible Shape

**Keywords:** Escher, Penrose, Illusion, Paradox, Mystery.

**Description:**
Shapes like the Penrose Triangle that cannot exist in 3D space but can be drawn in 2D.

## Usage

### 1. SVG Manual Drawing
Draw the shape using fixed paths that line up perfectly to create the illusion.

### 2. CSS 3D "Broken" Perspective
Arrange 3D transformed elements such that from *one specific camera angle*, they align to form an impossible object.
