# Spiral Vortex

**Keywords:** Hypnosis, Spin, Tunnel, Vertigo, Twist.

**Description:**
Lines or shapes twisting into a center point.

## Usage

### 1. CSS Conic Gradient
A repeating conic gradient animation.

```css
.spiral {
  background: repeating-conic-gradient(from 0deg, black 0deg, black 10deg, white 10deg, white 20deg);
  animation: spin 5s linear infinite;
  border-radius: 50%;
}
```
