# Wave Interference

**Keywords:** Physics, Ripple, Overlap, Moiré, Math.

**Description:**
Two or more wave patterns overlapping to create constructive and destructive interference patterns.

## Usage

### 1. WebGL / Canvas
Sum multiple sine waves.
`Height = sin(dist(p, source1)) + sin(dist(p, source2))`
Map height to color (black/white).
