# Avalanche Cloud

**Keywords:** Snow, Debris, Falling, Rolling, White.

**Description:**
A distinctive billowing cloud of white powder moving down a slope.

## Usage

### 1. Sine Wave Smoke
Multiple overlapping large semi-transparent white circles moving down and outwards, expanding in size.

### 2. Metatballs (Gooey Filter)
Using the CSS filter hack to merge the smoke particles into a continuous mass.
