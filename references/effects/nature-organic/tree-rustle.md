# Tree Rustle

**Keywords:** Leaves, Canopy, Wind, Shaking, Forest.

**Description:**
The individual bunches of leaves on a tree moving slightly independent of each other.

## Usage

### 1. CSS Transform
Separate the tree into layers (branches/leaf clumps) and rotate them slightly with different phases.

```css
.branch-1 { animation: rustle 2s ease-in-out infinite alternate; }
.branch-2 { animation: rustle 3s ease-in-out infinite alternate; }
```
