# Butterfly Swarm

**Keywords:** Insects, Flight, Beauty, Nature, Chaotic.

**Description:**
Colorful butterflies fluttering in a group, often following a target or moving erratically.

## Usage

### 1. Image Sprite + Motion Path
Animate a sprite sheet of wings flapping. Move the container along a bezier curve.

### 2. Flocking Algorithm (Boids)
Use Separation, Alignment, and Cohesion rules to make them move as a group.
