# Permission Error

**Keywords:** 403, Access Denied, Paywall.

**Description:**
User is connected but not allowed to see content.

## Usage

### 1. Login Prompt
If logged out.

### 2. Upgrade Prompt
If feature locked.

```html
<div class="lock-screen">
  <icon>🔒</icon>
  <h3>Pro Feature</h3>
  <button>Upgrade Now</button>
</div>
```
