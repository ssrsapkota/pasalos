# Inline Error

**Keywords:** Validation, Field Error, Form Feedback.

**Description:**
Errors specific to a single input field.

## Usage

### 1. Color & Icon
Don't rely on color alone.

```css
.input-error {
  border-color: red;
}
.error-msg {
  color: red;
  font-size: 0.8em;
  display: flex;
  align-items: center;
  gap: 0.5em; /* Space for icon */
}
```
