# Empty Illustration

**Keywords:** Zero Data, Blank Slate, Graphic.

**Description:**
A friendly, branded illustration used when there is no data to show. Prevents the page from feeling "broken" or stark.

## Usage

### 1. Centered Hero
Large illustration in the middle of the empty container.

```css
.empty-state {
  display: flex;
  flex-direction: column;
  align-items: center;
  text-align: center;
  padding: 4rem 2rem;
}
```
