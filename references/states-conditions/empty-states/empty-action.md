# Empty Action

**Keywords:** CTA, Call to Action, Button.

**Description:**
The primary button presented in an empty state to help the user populate the screen (e.g., "Create Item", "Add Friend").

## Usage

### 1. Primary Button
Make it the most prominent element.

```html
<div class="empty-state">
  <img src="empty-box.svg" />
  <p>You haven't added any items.</p>
  <button class="btn-primary">Add New Item</button>
</div>
```
