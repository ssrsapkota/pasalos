# Feature Discovery

**Keywords:** New, Highlight, Beacon.

**Description:**
Drawing attention to a specific new element in the UI without interrupting workflow.

## Usage

### 1. Pulsing Dot (Beacon)
Subtle attractor.

```css
.beacon {
  width: 10px; height: 10px;
  background: blue;
  border-radius: 50%;
  animation: pulse 2s infinite;
}
```
