# Character Design

> **Mascots and people**

Characters humanize a brand.

## Principles
-   **Proportions:** exaggerated heads/limbs (cute/friendly) vs realistic (serious/professional).
-   **Diversity:** ensure characters represent different skin tones, hair types, and non-binary options.
-   **Expressions:** Build a library of emotions (Success, Error, Thinking, Waiting).

## Usage
-   **Onboarding:** A guide character (mascot) walking the user through features.
-   **Empty States:** A character looking confused or searching for data.
