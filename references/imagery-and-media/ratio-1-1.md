# 1:1 Aspect Ratio

> **Square for avatars, thumbnails**

The safest ratio for responsive design as it never changes shape.

## Usage
-   **Avatars:** Almost exclusively square (often masked to circle).
-   **E-commerce Grids:** Uniform product listings.
-   **Instagram:** The classic social media feed format.

## Calculation
-   `height = width`
-   CSS: `aspect-ratio: 1 / 1;`
