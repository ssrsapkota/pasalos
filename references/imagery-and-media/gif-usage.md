# GIF Usage

> **When animated GIFs work**

The `.gif` format is ancient and inefficient, but universally supported.

## Use Cases
-   **Email:** The only reliable way to show animation in email newsletters.
-   **Documentation:** Quick, silent screencasts of a bug or feature.
-   **Social:** Memes and reactions.

## Avoid
-   **Hero Backgrounds:** Use MP4/WebM instead (GIFs form massive file sizes for HD video).
-   **UI Animations:** Use Lottie/SVG/CSS instead (GIFs have jagged edges/no alpha transparency).
