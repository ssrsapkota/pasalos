# Video Thumbnails

> **Preview image selection**

The static image shown before the video plays (Poster image).

## Choice
-   Don't use the first frame (often black).
-   Choose a frame that represents the most exciting or informative part of the video.
-   **Overlay:** Often superimpose a "Play" icon so users know it's a video, not an image.

## Loading
-   Load the poster image immediately (lightweight), lazy load the video player JS.
