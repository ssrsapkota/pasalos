# Illustration Style

> **Flat, dimensional, textured**

Illustration adds personality and clarifies abstract concepts.

## Styles
-   **Flat:** Modern, clean, scalable (vectors). Good for SaaS.
-   **Dimensional:** Uses shadows, gradients, and isometric angles to add depth.
-   **Textured:** Adds grain, brush strokes, or imperfections for a human/organic feel.
-   **Line Art:** Matches outlined icons for a technical or elegant look.

## Consistency
-   Maintain a consistent stroke width and color palette across all illustrations.
-   Don't mix "Clip Art" styles with "Hand Drawn" styles on the same site.
