# Spot Illustrations

> **Small decorative graphics**

Slightly larger and more detailed than icons, but smaller than hero illustrations.

## Usage
-   **Feature Lists:** Accompanying headers to explain a benefit.
-   **Empty States:** "No messages found".
-   **Cards:** Adding visual interest to a dashboard widget.

## Size
-   Typically 64px - 128px.
-   Should be legible without taking over the screen.
