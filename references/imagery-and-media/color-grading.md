# Color Grading

> **Consistent photo filtering**

Applying a consistent color lookup (LUT) or filter to all images.

## Techniques
-   **Desaturation:** Reduces color clash, creating a sleeker, more uniform look.
-   **Tinting:** Adding a subtle brand color cast (e.g., a slight blue tint for a tech company).
-   **Contrast Boost:** Making blacks deeper and whites brighter for a punchy, modern feel.

## Automation
-   Use CSS filters (sepia, saturate, contrast) for basic grading in the browser.
-   Use server-side image processing (Cloudinary, Imgix) for advanced LUTs.
