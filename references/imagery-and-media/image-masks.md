# Image Masks

> **Shaped image containers**

Breaking out of the rectangle.

## Shapes
-   **Circle:** Avatars.
-   **Rounded Rect:** Soft UI.
-   **Organic Blobs:** Friendly, creative (using `clip-path` or SVG masks).
-   **Diagonal/Slant:** Dynamic, energetic.

## CSS Clip-path
-   `clip-path: circle(50%)`
-   `clip-path: polygon(...)`
-   Allows for complex shapes without editing the original image file.
