# Transparency Layers

**Keywords:** Depth, Stacking, Opacity.

**Description:**
Stacking multiple semi-transparent layers to create depth and visual interest without using shadows.

## Usage

### 1. Alpha Stacking
Overlapping shapes.

```css
.circle-1 { background: rgba(255, 0, 0, 0.5); }
.circle-2 { background: rgba(0, 0, 255, 0.5); }
/* Overlap creates purple */
```
