# Linear Gradient

**Keywords:** Directional, Fade, Smooth Transition.

**Description:**
The most common gradient type, transitioning colors along a straight line. Good for backgrounds, buttons, and overlays.

## Usage

### 1. Basic Direction
Top to bottom is default.

```css
.bg {
  background: linear-gradient(to right, #000, #fff);
}
```

### 2. Angled
Specific degrees.

```css
.bg-angled {
  background: linear-gradient(45deg, blue, red);
}
```
