# Gradient Angle

**Keywords:** Degrees, Turn, Direction.

**Description:**
Precisely controlling the angle of the gradient line.

## Usage

### 1. Degrees
Standard rotation (0deg = Bottom to Top).

```css
background: linear-gradient(135deg, blue, red);
```

### 2. Turns
0.25turn = 90deg.

```css
background: linear-gradient(0.25turn, blue, red);
```
