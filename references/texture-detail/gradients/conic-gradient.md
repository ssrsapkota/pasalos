# Conic Gradient

**Keywords:** Sweep, Pie Chart, Color Wheel.

**Description:**
Colors rotated around a center point. Great for pie charts, color wheels, or metallic reflections.

## Usage

### 1. Border Glow Effect
Creating a rotating border.

```css
.border-wrap {
  background: conic-gradient(from 0deg, red, blue, red);
}
```

### 2. Pie Chart
Hard stops.

```css
.pie {
  background: conic-gradient(red 0% 25%, blue 25% 100%);
  border-radius: 50%;
}
```
