# Geometric Pattern

**Keywords:** Triangles, Hexagons, Shapes.

**Description:**
Repeating geometric shapes creating a tessellation effect.

## Usage

### 1. SVG Background (Recommended)
Inline SVG for complex shapes.

```css
.triangles {
  background-image: url("data:image/svg+xml,...");
}
```

### 2. CSS Conic Gradient (Hexagons)
Very complex to do with pure CSS, usually avoid unless simple.
