# Line Pattern

**Keywords:** Stripes, Diagonal, Pinstripe.

**Description:**
Repeating lines. Can be horizontal, vertical, or diagonal.

## Usage

### 1. Diagonal Stripes
CSS repeating linear gradient.

```css
.stripes {
  background: repeating-linear-gradient(
    45deg,
    #fff,
    #fff 10px,
    #eee 10px,
    #eee 20px
  );
}
```
