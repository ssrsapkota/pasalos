# Inner Shadow

**Keywords:** Inset, Dent, Engraved.

**Description:**
Using `inset` shadows to make elements look pressed into the surface.

## Usage

### 1. Pressed Button
Simulating a button being held down.

```css
.pressed {
  box-shadow: inset 2px 2px 5px rgba(0,0,0,0.2);
}
```

### 2. Input Fields
Classic "well" look.

```css
input {
  box-shadow: inset 0 1px 3px rgba(0,0,0,0.1);
}
```
