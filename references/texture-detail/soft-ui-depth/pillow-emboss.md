# Pillow Emboss

**Keywords:** Soft, Puffy, Tactile.

**Description:**
A variation of neumorphism that looks softer and more "puffy", like a pillow.

## Usage

### 1. Soft Emboss
Combining multiple shadows/highlights.

```css
.pillow {
  box-shadow:
    inset 2px 2px 5px rgba(255,255,255,0.7),
    inset -2px -2px 5px rgba(0,0,0,0.1),
    5px 5px 10px rgba(0,0,0,0.1);
}
```
