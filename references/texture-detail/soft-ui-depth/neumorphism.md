# Neumorphism

**Keywords:** Soft UI, Extruded, Plastic.

**Description:**
A visual style using two shadows (one light, one dark) to make elements appear to be extruded from the background material.

## Usage

### 1. The classic "Soft" shadow
Requires background and element to have the exact same color.

```css
.neumorph {
  background: #e0e0e0;
  box-shadow:  20px 20px 60px #bebebe,
              -20px -20px 60px #ffffff;
  border-radius: 50px;
}
```
