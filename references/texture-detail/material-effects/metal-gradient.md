# Metal Gradient

**Keywords:** Silver, Gold, Brushed Metal, Chrome.

**Description:**
Using gradients to simulate the specular highlights and reflections of metallic surfaces.

## Usage

### 1. Brushed Steel
Linear gradient with sharp transitions.

```css
.metal {
  background: linear-gradient(
    to bottom,
    #888 0%,
    #ddd 50%,
    #888 100%
  );
}
```

### 2. Chrome
More complex reflection.

```css
.chrome {
  background: linear-gradient(
    to bottom,
    #333 0%,
    #fff 50%,
    #888 51%,
    #fff 100%
  );
}
```
