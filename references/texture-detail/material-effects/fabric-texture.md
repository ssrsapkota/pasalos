# Fabric Texture

**Keywords:** Cloth, Denim, Canvas, Woven.

**Description:**
A texture resembling woven threads. Adds a distinct tactile feel.

## Usage

### 1. CSS Pattern
Repeating linear gradients.

```css
.fabric {
  background-color: #333;
  background-image:
    linear-gradient(45deg, rgba(0,0,0,0.2) 25%, transparent 25%, transparent 75%, rgba(0,0,0,0.2) 75%),
    linear-gradient(45deg, rgba(0,0,0,0.2) 25%, transparent 25%, transparent 75%, rgba(0,0,0,0.2) 75%);
    background-size: 4px 4px; /* Tight weave */
}
```
