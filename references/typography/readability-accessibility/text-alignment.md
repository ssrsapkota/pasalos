# Text Alignment

**Keywords:** Left, Center, Right, Justify, Rag.

**Description:**
Alignment controls how the text is anchored. In Western languages (LTR), the eye naturally seeks the left edge as a starting point for each line.

## Guide

### 1. Left Align (Ragged Right)
**The Default & The Best.**
*   Consistent starting point for the eye.
*   Uneven right edge (the "rag") creates variety and prevents spacing issues.
*   *Use for:* 99% of body text.

### 2. Center Align
**Decorative / Formal.**
*   Hard to read for long blocks because the starting point of every line changes, forcing the eye to "hunt" for it.
*   *Use for:* Headlines, Poems, Short invitations, Captions (2-3 lines max).

### 3. Right Align
**Niche.**
*   Hard to read.
*   *Use for:* Side notes, table data (numbers), specific UI labels.

### 4. Justify
**Problematic on Web.**
*   Forces straight edges on both sides.
*   Causes variable word spacing ("rivers").
*   *Use for:* Only if you have a sophisticated hyphenation engine and wide columns. Avoid on mobile.
