# Weight Scale

**Keywords:** Thin, Regular, SemiBold, Black, 100-900.

**Description:**
Font weight refers to the thickness of the character strokes. A complete weight scale provides flexibility for creating hierarchy.

## The Standard Scale

*   **100 (Thin/Hairline):** Use only at HUGE sizes. Invisible at body size.
*   **200 (Extra Light):** Elegant, delicate.
*   **300 (Light):** Contemporary, clean.
*   **400 (Regular/Normal):** The default body weight.
*   **500 (Medium):** Slight emphasis, good for dark mode body text.
*   **600 (SemiBold):** Subheadings, buttons, UI states.
*   **700 (Bold):** Standard bold, headlines.
*   **800 (Extra Bold):** Heavy impact.
*   **900 (Black):** Maximally loud, decorative.

## Usage Tips

### 1. Skip a Step
To create visible contrast, skip a weight class.
*   Regular (400) vs Medium (500) -> Too subtle.
*   Regular (400) vs SemiBold (600) -> Clear contrast.

### 2. Dark Mode
Light text on a dark background appears thicker (optical illusion).
*   **Solution:** Use a lighter weight (e.g., 300) or reduce opacity.
*   **Avoid:** Using 900 weight in pure white on pure black (causes halation).
