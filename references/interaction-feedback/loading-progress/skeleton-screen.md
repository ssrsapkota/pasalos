# Skeleton Screen

**Description:**
A blank version of the page into which information is gradually loaded. It mimics the structure of the content (rectangles for text, circles for avatars).

**Key Principles:**
- **Perceived Speed**: Makes the app feel faster because the structure appears instantly.
- **Layout Stability**: Prevents layout shifts (CLS) by reserving space for content before it arrives.

**Common Patterns:**
- **Grey Boxes**: Simple grey shapes matching the content dimensions.
- **Pulsing**: The shapes pulse opacity to indicate activity.
