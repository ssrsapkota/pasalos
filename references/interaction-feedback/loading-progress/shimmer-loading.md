# Shimmer Loading

**Description:**
A specific animation style for skeleton screens where a lighter band of color moves across the grey boxes, mimicking light reflection.

**Key Principles:**
- **Direction**: Usually moves from left to right.
- **Speed**: Should be fast enough to look active (e.g., 1.5s - 2s loop) but not frantic.

**Common Patterns:**
- **CSS Gradient**: Using linear-gradient and background-position animation.
- **Masks**: Using an SVG mask to create the shimmer effect.
