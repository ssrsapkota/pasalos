# Press Depth

**Description:**
Reducing the "elevation" (shadow) of an element to zero or a very low value, stimulating the effect of the element being pushed flat against the surface.

**Key Principles:**
- **Shadow Removal**: If a button has a shadow on rest or hover, remove or shrink it significantly on press.
- **Offset Change**: If using a hard shadow (retro/brutalist style), shift the element to cover the shadow, reducing the perceived distance.

**Common Patterns:**
- **Flat Press**: Material Design buttons lose their elevation shadow.
- **3D Button**: "Neumorphic" or retro 3D buttons visually depress into the screen.
