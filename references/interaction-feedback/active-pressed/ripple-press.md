# Ripple Press

**Description:**
The material design "ripple" effect where a wave of color spreads outward from the point of contact.

**Key Principles:**
- **Origin**: The ripple should start exactly where the user clicked/tapped.
- **Containment**: The ripple should be clipped to the bounds of the element.
- **Dissipate**: The ripple should fade out smoothly as the press is released or the animation completes.

**Common Patterns:**
- **Material Buttons**: The classic Android/Google ripple.
- **List Items**: Clicking a row in a list triggers a ripple.
