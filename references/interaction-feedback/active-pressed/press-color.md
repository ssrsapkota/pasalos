# Press Color

**Description:**
Changing the color (usually darkening) when an element is pressed. This mimics the effect of a surface moving away from the light source or becoming denser.

**Key Principles:**
- **Contrast**: Ensure the pressed state is distinct from the hover state. Usually, Hover is slightly darker/lighter, and Press is even darker/lighter.
- **Feedback**: Provides immediate confirmation of the action.

**Common Patterns:**
- **Darker Shade**: 10% darker than the hover state.
- **Desaturation**: Sometimes reducing saturation can indicate a "depressed" state.
- **Background Fill**: Outline buttons might fill with a solid color on press.
