# Focus Within

**Description:**
Styling a parent container when one of its children has focus. This is powerful for complex components like forms or search bars.

**Key Principles:**
- **Context**: Helps the user understand which larger component they are interacting with.
- **Simplicity**: Don't overdo it; if the parent lights up, the child might not need an intense focus ring of its own, or they can work in tandem.

**Common Patterns:**
- **Form Groups**: Highlighting the entire input group (label + input + icon) when the input is focused.
- **Search Bar**: A search icon and wrapper lighting up when the text input inside is focused.
- **Navigation Lists**: Highlighting a whole list item when a link inside it is focused.
