# Focus Color

**Description:**
Using color to indicate focus, often aligning with the primary brand color to create a cohesive experience.

**Key Principles:**
- **Brand Consistency**: The focus color should usually be your primary brand color (e.g., if your brand is blue, use a blue focus ring).
- **Semantics**: For error states, the focus ring might turn red to indicate attention is needed on an invalid field.

**Common Patterns:**
- **Theme Matching**: The input border changes to the primary color on focus.
- **Accents**: A glow matching the brand's accent color.
