# Skip Links

**Description:**
Hidden links that become visible only when focused, allowing keyboard users to "skip" repetitive content (like navigation menus) and jump directly to the main content.

**Key Principles:**
- **Invisibility**: Hidden by default (usually pushed off-screen or zero opacity).
- **Visibility on Focus**: Must appear immediately when it receives focus (tab into page).
- **Placement**: Should be the very first tab-able element on the page.

**Common Patterns:**
- **"Skip to Content"**: The standard link text.
- **Top Bar**: Animate down from the top of the viewport when focused.
