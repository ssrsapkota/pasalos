# Hover Cursor

**Description:**
Changing the mouse cursor icon to indicate the type of interaction available. This is a fundamental browser feature but can be customized for richer apps.

**Key Principles:**
- **Standardization**: Stick to standard cursors (`pointer`, `text`, `default`, `grab`, `not-allowed`) for standard actions to avoid confusion.
- **Custom Cursors**: Use custom SVG cursors sparingly, mostly for immersive experiences (games, canvas apps) or specific tools (e.g., a "pen" tool cursor).
- **Feedback**: The cursor change happens instantly, providing the fastest possible feedback loop.

**Common Patterns:**
- **Pointer**: The hand icon is standard for clickable links and buttons.
- **Grab/Grabbing**: Open hand for draggable items, closed hand while dragging.
- **Zoom In/Out**: Magnifying glass for image galleries.
- **Wait/Progress**: Spinner cursor for background processes (use sparingly).
