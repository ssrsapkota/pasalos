# Hover Elevation

**Description:**
Using shadow and z-index changes to create the illusion of an element lifting off the page when hovered. This mimics the physical action of picking something up or bringing it closer.

**Key Principles:**
- **Light Source**: Shadows should be consistent with a global light source.
- **Diffusion**: As an element lifts (higher elevation), the shadow should become larger and more blurred (diffused), but lighter in opacity.
- **Motion**: Pair with a slight scale transform to enhance the lifting effect.

**Common Patterns:**
- **Card Lift**: A card component gains a larger, deeper shadow on hover.
- **Button Rise**: A flat or slightly raised button appears to float higher.
- **Floating Action Button (FAB)**: intense elevation change to signify primary action.
