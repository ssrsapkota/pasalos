# Celebration

**Description:**
Delightful animations to reward the user for a significant accomplishment.

**Key Principles:**
- **Rare**: Don't use it for everything. Save it for big moments (e.g., reaching "Inbox Zero", finishing a course).
- **Joyful**: Use bright colors and fun motion.

**Common Patterns:**
- **Confetti**: Particles raining down.
- **Mascot**: A character giving a thumbs up.
- **Sound**: A subtle "ding" (optional and mutable).
