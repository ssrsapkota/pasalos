# Success State

**Description:**
Visual feedback confirming that an action completed successfully.

**Key Principles:**
- **Positive Color**: Green is the universal standard for success.
- **Iconography**: A checkmark symbol is universally understood.
- **Timing**: Should appear immediately after the action completes (or optimistically).

**Common Patterns:**
- **Green Text/Border**: Changing input color to green.
- **Checkmark Icon**: Appearing next to or inside the component.
- **Message**: "Saved successfully" text.
