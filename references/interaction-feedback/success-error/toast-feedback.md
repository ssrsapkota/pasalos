# Toast Feedback

**Description:**
A temporary notification that appears autonomously (usually bottom-right or top-center) to confirm an action.

**Key Principles:**
- **Non-blocking**: Users should still be able to interact with the rest of the page.
- **Auto-dismiss**: Should disappear after a few seconds (e.g., 5s).
- **Undo**: A "Undo" action inside the toast is a powerful pattern (e.g., "Email deleted. [Undo]").

**Common Patterns:**
- **Snackbar**: A dark strip at the bottom of mobile screens.
- **Notification Stack**: Multiple toasts stacking on top of each other.
