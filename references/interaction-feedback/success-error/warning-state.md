# Warning State

**Description:**
Indicating a potential issue that isn't a hard stop or error, but requires attention.

**Key Principles:**
- **Cautionary Color**: Yellow or amber is standard.
- **Non-Blocking**: The user can usually proceed, but should proceed with caution.

**Common Patterns:**
- **Password Strength**: "Weak" password indicator.
- **Disk Space**: "Storage almost full" warning.
- **Destructive Action**: Warning before deleting something ("Are you sure?").
