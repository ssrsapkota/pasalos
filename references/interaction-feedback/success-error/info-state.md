# Info State

**Description:**
Providing neutral interaction or documentation that helps the user understand context.

**Key Principles:**
- **Neutral Color**: Blue is standard, or greys.
- **Supportive**: Should not interrupt the user flow, but offer help if noticed.

**Common Patterns:**
- **Blue Banner**: A blue strip at the top of a page ("Update available").
- **Info Icon**: 'i' in a circle tooltip.
