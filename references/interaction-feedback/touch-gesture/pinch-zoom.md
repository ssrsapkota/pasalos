# Pinch to Zoom

**Description:**
Using two fingers to expand (zoom in) or contract (zoom out) content.

**Key Principles:**
- **Fluidity**: The content should scale in real-time with the finger movement (1:1 tracking).
- **Resolution**: Ensure the content (images/vectors) stays sharp when zoomed.

**Common Patterns:**
- **Images**: Zooming into photos in a gallery.
- **Maps**: Zooming into a map view.
