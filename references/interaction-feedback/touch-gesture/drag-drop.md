# Drag and Drop

**Description:**
Moving an object from one location to another by grabbing it, dragging it, and releasing it.

**Key Principles:**
- **Affordance**: The element should look draggable (e.g., using a "grip" icon of 6 dots).
- **Drop Targets**: Highlight valid drop zones when dragging starts to guide the user.
- **Ghost**: Show a semi-transparent version (ghost) of the item attached to the cursor/finger while dragging.

**Common Patterns:**
- **Reordering**: Changing the order of items in a list.
- **Kanban**: Moving cards between columns.
- **File Upload**: Dragging files from desktop to browser.
