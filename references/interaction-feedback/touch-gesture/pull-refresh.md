# Pull to Refresh

**Description:**
Dragging down from the top of a list to reload the content.

**Key Principles:**
- **Resistance**: The drag should feel rubber-bandy (increasing resistance).
- **Indicator**: A spinner or icon should appear to show how far needed to pull.

**Common Patterns:**
- **Standard**: Circle spinner appearing at the top.
- **Creative**: Animations related to the brand (e.g., a rocket launching) when the pull threshold is reached.
