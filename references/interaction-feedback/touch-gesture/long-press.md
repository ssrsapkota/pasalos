# Long Press

**Description:**
Pressing and holding an element for a set duration (e.g., 500ms) to trigger a secondary action.

**Key Principles:**
- **Feedback**: There should be a visual cue (like a growing circle or progress bar) indicating that a long press is being registered.
- **Vibration**: Haptic feedback (vibration) on success is very helpful on mobile.

**Common Patterns:**
- **Context Menu**: Opening a context menu (like right-click on desktop).
- **Selection Mode**: Entering a multi-select mode in a list.
