# Disabled Cursor

**Description:**
Changing the cursor to `not-allowed` to explicitly show that an interaction is forbidden.

**Key Principles:**
- **Clarity**: The `not-allowed` (circle with slash) cursor is universally understood.
- **Scope**: Apply it only when the user *would* expect to click but cannot.

**Common Patterns:**
- **Form Submit**: Hovering a disabled submit button shows the not-allowed cursor.
- **Restricted Areas**: Hovering over locked content.
