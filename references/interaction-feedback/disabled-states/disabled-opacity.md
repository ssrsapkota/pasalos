# Disabled Opacity

**Description:**
Reducing the opacity of an element to indicate it is not interactive. This is the most common way to signify a disabled state.

**Key Principles:**
- **readability**: Maintain enough contrast so the text is still readable, even if disabled.
- **Standard**: 0.5 (50%) opacity is a common standard.

**Common Patterns:**
- **Faded Button**: The entire button (text + background) becomes semi-transparent.
- **Ghosted Icon**: Icons become faint.
