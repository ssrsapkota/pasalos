# Disabled Color

**Description:**
Changing the color scheme to greyscale or muted tones to indicate inactivity.

**Key Principles:**
- **Desaturation**: Removing color communicates "life has left this element".
- **Greys**: Using neutral greys for backgrounds and borders.

**Common Patterns:**
- **Greyed Out**: Primary buttons turn grey.
- **Muted Text**: Text color changes from black/dark to a lighter grey.
